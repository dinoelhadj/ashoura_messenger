# Ashoura
## _Simple Chat APP_

Ashoura is a simple Android messenger app made by Java, that barely works.

## Features

- Login/Register using Firebase Authentication
- User Presence System
- Text messages
- Online friends (x)
- Add friends (x)
- Search (x)
- Profile (x)
